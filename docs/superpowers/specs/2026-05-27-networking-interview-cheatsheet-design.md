在 Jekyll 博客上发布一篇中文技术文章，聚焦计算机网络传输层与应用层的面试高频考点，以速查手册风格呈现，帮助作者复习准备后端/Java 实习面试。

### 内容大纲

#### 一、传输层

| 知识点 | 覆盖内容 |
|--------|----------|
| TCP vs UDP | 连接性、可靠性、有序性、速度、头部开销、适用场景对比 |
| TCP 三次握手 | SYN/ACK 流程、为什么是三次 |
| TCP 四次挥手 | 挥手流程、TIME_WAIT 作用 |
| TCP 可靠传输 | ACK、超时重传、滑动窗口、流量控制、拥塞控制(慢启动/拥塞避免/快重传/快恢复) |
| UDP | 无连接、不可靠、低延迟、典型应用场景 |

#### 二、应用层

| 知识点 | 覆盖内容 |
|--------|----------|
| HTTP vs HTTPS | 明文 vs 加密、端口、SSL/TLS 握手 |
| HTTP 状态码 | 1xx-5xx 分类 + 高频状态码速记 |
| HTTP 版本演进 | 1.0→1.1→2.0→3.0 关键变化 |
| GET vs POST | 语义、安全性、幂等性等区别 |
| Cookie vs Session | 存储位置、安全性、生命周期 |
| DNS 解析 | 递归+迭代查询流程 |
| URL 到页面渲染 | 完整链路综合题 |
